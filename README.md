# Тестовое задание — 3SNET Events Widget (Playwright + Python)


Репозиторий содержит пример автотеста для страницы: `https://dev.3snet.info/eventswidget/`.


## Что проверяет тест

- Страница доступна и загружается.
- На странице видны ключевые секции (шаги 1-4, кнопка "Сгенерировать превью", превью-блок, поле с кодом).
- Можно изменить параметры: выбрать опции в выпадающих списках (если присутствуют), задать ширину/высоту, переключить цветовую тему.
- При нажатии `Сгенерировать превью` создаётся `<iframe>` в превью-блоке.
- В поле с кодом появляется корректный HTML с `<iframe>` и параметрами `width`/`height`.
- В консоли браузера нет критических ошибок (уровень `error`).


## Требования

- Python 3.9+
- Playwright
- pytest


## Установка

```bash
# 1. Клонировать репозиторий
# git clone <репозиторий>
# cd 3snet


# 2. Создать виртуальное окружение (рекомендовано)
python -m venv .venv
# активация:
# Windows: .\.venv\Scripts\activate
# macOS/Linux: source .venv/bin/activate


# 3. Установить зависимости
pip install -r requirements.txt


# 4. Установить браузеры Playwright
playwright install


# 5. Запуск теста:
pytest -v
